# Android-VoiceAssistant
## 简介
仿照Siri，利用[百度语音](http://yuyin.baidu.com/)和[图灵机器人](http://www.tuling123.com/)，实现打电话，聊天等功能

## 功能
1. 给通讯录中联系人打电话
2. 给通讯录中联系人发短信
3. 打开应用
4. 网上查找资料
5. 聊天

## 注意事项
项目里的百度语音的APPId、APPKey、SecretKey和图灵机器人的APIKey只是为了让项目正常运行，实际使用过程中请替换成自己的实际值。
